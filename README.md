# AlgoViz_AI

A visualization tool for algorithm learning and understanding using AI-powered animations.

## Project Structure

- **backend/**: Python FastAPI backend with Celery workers for algorithm visualization
- **frontend/**: TypeScript React frontend for the user interface

## Getting Started

### Backend Setup
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## Features

- Interactive algorithm visualizations
- Support for sorting algorithms (Bubble Sort, Merge Sort, Selection Sort, etc.)
- AI-powered commentary and explanations
- Real-time animation rendering
